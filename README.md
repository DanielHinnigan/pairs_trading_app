# Pairs Trading Platform
A local web-based platform used for simple analysis of pairs of stocks. This project was developed as a personal project.

## Prerequisities
- Python Version: 3.13
- uv package manager installed. Can be installed using "pip install uv"

## Installation & Running
- Clone the repository
- Install the dependencies using "uv sync" in the terminal

## Launching the application
Use "uv run streamlit run main.py" to run the application
